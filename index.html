How it works

A daily operating loop for your AI chief of staff.

The skills are designed to run on schedules, but each can also be triggered manually. The point is not to chat more. The point is to make sure the obvious work gets done before you notice it.

01

Prepare

Pull calendar, attendee context, email history, past meeting notes, LinkedIn, and company research before meetings start.

02

Capture

Extract commitments from meeting transcripts, create Todoist tasks, detect duplicates, and draft the follow-ups you promised.

03

Surface

Send a concise executive digest with overdue work, stalled threads, pending intros, calendar risks, and drafts waiting for review.

Included skills

Fourteen practical skills that map to real EA and ECA jobs.

Each skill is just markdown, so you can read it, edit it, fork it, and make the behavior match how you actually work.

executive-control-system

ECA / ECS

Control-system layer for more time, less stress, and more control across inbox, calendar, loops, decisions, and stress load.

  • Draft-only authority boundaries
  • Decision compression
  • Open-loop radar
meeting-prep

Meeting prep

Briefings for today’s meetings with attendee research, RSVP flags, email context, past notes, and focus areas.

  • One message per meeting
  • Internal vs external logic
  • Pre-meeting reminders
action-items-todoist

Action items

Turns meeting commitments into Todoist tasks and draft emails, while preventing duplicate tasks from multiple cron runs.

  • Granola + Grain checks
  • Todoist task creation
  • Follow-up draft triggers
email-drafting

Email drafting

Draft-only Gmail automation for intros, scheduling intent, thanks, acknowledgements, and positive short replies.

  • Never sends automatically
  • Mirrors inbound language
  • Uses your style guide
executive-digest

Executive digest

A daily attention report across tasks, calendars, stalled email threads, promised follow-ups, and pending drafts.

  • Suppresses repeats
  • Checks both Gmail accounts
  • Flags calendar risks
todoist-due-drafts

Due-today drafts

Looks at today’s and overdue outreach tasks, creates contextual Gmail drafts, and notifies you for review.

  • Task intent filtering
  • Thread-aware drafts
  • Existing-draft detection
daily-inbox-sweep

Daily inbox sweep

Processes an assistant inbox, organizes new items, routes tasks, dedupes alerts, and flags human attention items.

  • Categorizes new items
  • Routes tasks safely
  • Dedupes alerts
daily-standup-prep

Daily standup prep

Creates tight morning standups and EOD closeouts from recent work signals, task state, and blockers.

  • Yesterday / Today / Blockers
  • EOD closeout mode
  • No invented recaps
weekly-life-admin

Weekly life admin

Reviews upcoming logistics, renewals, errands, appointments, and week-ahead preparation.

  • 7-day calendar scan
  • Renewal checks
  • Day-by-day checklist
weekly-competitor-scan

Competitor scan

Tracks AI assistant and productivity market moves, opportunities, and threats with cited sources.

  • Market moves
  • Opportunities
  • Threats
weekly-content-generation

Content generation

Turns recent work, meetings, and notes into weekly social posts, blog outlines, and newsletter drafts.

  • Uses real context
  • Voice-aware drafts
  • No invented claims
monthly-bill-review

Monthly bill review

Reviews expenses and subscriptions, flags anomalies, and creates cancel/dispute/investigate action items.

  • Subscription review
  • Anomaly flags
  • Read-only by default
quarterly-review

Quarterly review

Builds a one-page executive review of goals, metrics, wins, misses, patterns, and next-quarter priorities.

Humanizer

Setup

Clone, configure, load, schedule.

The repository includes a full setup guide and ready-to-adapt cron examples. Personal config stays local in config/user.json, which is gitignored.

Step 01

Clone

Put the repo at ~/executive-assistant-skills so relative skill paths resolve correctly.

Step 02

Configure

Copy the example config and fill in email accounts, timezone, Slack, WhatsApp, and workspace paths.

Step 03

Connect tools

Authenticate Gmail/Calendar through gog, meeting tools through mcporter, and Todoist through the CLI.

Step 04

Schedule

Add cron jobs for meeting prep, action items, due-today drafts, digest, and token refresh.

git clone https://github.com/mgonto/executive-assistant-skills.git ~/executive-assistant-skills
cd ~/executive-assistant-skills
cp config/user.example.json config/user.json
# Then follow docs/setup.md and docs/crons.md
